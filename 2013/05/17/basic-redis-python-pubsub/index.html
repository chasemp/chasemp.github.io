basic redis python pubsub

Friday, May 17, 2013 » pubsub, python, redis

Preamble

I am serializing a dict as JSON and publishing it. The subscriber will
receive and print. It is not necessary to use JSON as redis has awesome
datatype compatibility with python: http://redis.io/topics/data-types.

Using pubsub to distribute messages. Often the 'right way'.

This is using python-redis from the debian repos:

http://packages.debian.org/wheezy/python-redis

Publishing an event:

This is just an example.


    import redis
    import time
    import json
    r_server = redis.Redis("redis", db=1)
    payload = {}
    payload['event'] = 'played with legos'
    payload['ts'] = time.time()
    json_values = json.dumps(payload, separators=(',', ':'))
    r_server.publish('kidsgames', json_values)

Subscribing to events


    import redis
    import time
    import json
    r_server = redis.Redis("redis", db=1)
    r_server.subscribe('kidsgames')
    while True:
        for m in r_server.listen():
            if m['type'] == 'subscribe':
                if m['data'] == 1:
                    log.info('subscribed to: %s' % (m['channel']))
            #this is what most pubsub submissions will match
            elif m['type'] == 'message':
                #our payload is in the data field of the response
                print json.loads(m['data'])
            else:
                print m
